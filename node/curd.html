1.如何使用SQL语句实现增加、修改和删除？

INSERT INTO 表明 (列1,...) VALUES (值1,...)

INSERT INTO `Articles` (`title`, `content`)
VALUES ('静夜思', '床前明月光，疑是地上霜。')

2.SQL语句的语法规则

大小写：这条SQL语句，都使用了大写字母改为小写也是一样可以正确运行的，只是大写的更规范一些而已。

反引号： 表的名字和字段的名字，规范来说最好加上`号。

引号：内容都是字符串，一定要加上引号。不加上引号的话，MySQL就无法正确的识别为字符串，也会报错的。当然，如果是数字的话，就不用打引号了。

修改语句

UPDATE 表名
SET 列1=值1,
    列2=值2, ...WHERE 条件

-- 例如：
UPDATE `Articles`
SET `title`='黄鹤楼送孟浩然之广陵',
    `content`='故人西辞黄鹤楼，烟花三月下扬州。'
WHERE `id` = 2;

删除语句

DELETE
FROM 表名
WHERE 条件

-- 例如：
DELETE
FROM `ARTICLES`
WHERE `id` = 5;
